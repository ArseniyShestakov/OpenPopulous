OpenPop Menus

Main Menu

When Populous starts it will play the full motion video (FMV) of the bullfrog logo. It then displays the main menu with the following items.

  • New Game -> Tutorial Confirmation -> New game (FMV) -> Solar System
  • Tutorial -> Straight to in-game tutorial
  • Load Game -> Load game screen -> Staight to in-game
  • Multiplayer -> Connection type -> Host/Join, game list -> Create Game (name and port) -> Multiplayer lobby
  • Options -> Game/Sound/Graphics
  • Rolling Demo -> Plays through demos in order
  • Credits -> Plays the credits
  • Coming Soon -> Plays Dungeon Keeper 2 demo
  • Quit

OpenPop will have a slightly changed menu as more features are added and there are some things taken away such as the trailer for Dungeon Keeper 2.
